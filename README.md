#  PNG to JPG Image Converter (Python GUI)

A simple Python GUI application that converts PNG images to JPG format using Tkinter and Pillow (PIL).

---

##  Features
- Easy-to-use graphical interface
- Select PNG images via file dialog
- Convert and save images as JPG
- Beginner-friendly Python project

---

##  Tech Stack
- Python
- Tkinter
- Pillow (PIL)
